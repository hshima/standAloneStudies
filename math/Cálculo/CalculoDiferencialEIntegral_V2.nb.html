Funções: noções básicas

Definição das variáveis de amostragem:

x <- c(-100:100)
y <- x

Exemplos de função:

  • Função reta: \(f(x) = x\)

  • Função cônica: \(f(x) = x^2\)

  • Função cúbica: \(f(x) = x^3\)

  • Função Exponencial: \(f(x) = e^x\)

  • Função cosseno x: \(f(x) = cos(x)\)

  • Função tangente x: \(f(x) = tg(x)\)

Definição de função

  • Nem toda curva representa uma função: Um corte paralelo em relação ao eixo y não pode ter mais de um ponto.

  • Sequências numéricas podem ser funções. Exemplo: \[ f : N \to N\\ n \mapsto m = f(n) \biggl\{ \begin{array}{rcl} \frac{n}{2} & \mbox{ se $n$ é par}\\ \frac{3n+1}{2} & \mbox{ se n é ímpar}\end{array} \]

Função exponencial

\[ f(x) = a^x \mbox{, onde } a>0 \mbox{ e } a \ne 1 \mbox{.}\\ \mbox{se } a > 1 \mbox{ a função é crescente,}\\ \mbox{se } a < 1 \mbox{, a função é decrescente.}\\ Dom(f) = \mathbb{R} \mbox{ e } Im(f) = (0, +\infty) \] se \[ a>1 \mbox{, a função é crescente } \]

Função logarítmica

\[ \mbox{se } y = f(x) = a^x \mbox{, então } x = f^-1(y)= \log_a{y} \]
